## Objetivo da Atividade: ## 
A atividade 2 tem como finalidade aplicar conceitos fundamentais de programação orientada a objetos, como uso de classes, atributos, métodos, construtores e manipulação de listas. <br>
Além disso, o sistema implementa as operações básicas de um CRUD, permitindo ao usuário cadastrar, visualizar, atualizar e remover livros.

## Instruções para Executar o Programa: ##
- O menu será exibido no terminal; <br>
- Digite o número correspondente à operação desejada e pressione Enter; <br>
- Siga as instruções exibidas na tela para interagir com o sistema.

## Exemplo da Saída do Programa no Terminal: ##
- O sistema exibe um menu com opções de cadastro, listagem, atualização, remoção e saída; <br>
- Ao escolher cadastrar livro, o usuário informa título, autor, número de páginas e ano, e o sistema confirma o cadastro. <br>
- Ao escolher listar livros, o sistema exibe os livros cadastrados. <br>
- Ao escolher atualizar livro, o usuário informa o título e altera os dados, recebendo confirmação. <br>
- Ao escolher remover livro, o usuário informa o título e o sistema remove o livro. <br>
- Ao escolher sair, o programa é encerrado.
