## Objetivo da Atividade: ## 
- A atividade 3 tem como finalidade desenvolver um aplicativo de calculadora simples utilizando Flutter. <br>
- Aplicar conceitos de interface gráfica com widgets.<br>
- Implementar operações matemáticas básicas (adição, subtração, multiplicação e divisão).<br>
- Organizar o código com componentes reutilizáveis.

## Instruções para Executar o Programa: ##
- A interface intuitiva da calculadora mostram um display com os números e as operações; <br>
- Digite o número, a operação desejada e o segundo número; <br>
- Por fim, o resultado é exibido ao clicar em " = ".

## Exemplo da Saída do Programa no Terminal: ##
- O aplicativo exibe uma interface de calculadora com botões numéricos e operações.<br>
- O usuário insere os números clicando nos botões.<br>
- Ao selecionar uma operação e pressionar " = ", o resultado é mostrado na tela.<br>
- O botão “Limpar” reinicia a operação e limpa o display.
