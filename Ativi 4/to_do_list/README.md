#  **Programação para Dispositivos Móveis**

## Nome: ##
Maria Clara Florente Dalapria <br>

## Objetivo da Atividade: ## 
- Desenvolver um aplicativo de lista de tarefas simples; <br>
- Aplicar conceitos de interface gráfica com widgets; <br>
- Utilizar componentes reutilizáveis para organização do código; <br>
- Implementar tarefas na lista e riscá-las da lista ou exclui-las.

## Instruções para Executar o Programa: ##
- Instalar o Flutter e configurar o ambiente de desenvolvimento; <br>
- Abrir o projeto no VS Code; <br>
- Executar o projeto com: flutter run.

## Exemplo da Saída do Programa no Terminal: ##
- O aplicativo exibe uma tela com lista de tarefas;<br>
- O usuário pode adicionar uma nova tarefa digitando um texto;<br>
- As tarefas são exibidas em formato de lista;<br>
- O usuário pode remover tarefas quando desejar.
